# Feed
    
- Feed流，进行新闻推送

|    字段    |   类型    |  空  | 默认 |   注释   |     |
| :--------- | :-------- | :--- | ---- | -------- | --- |
| title      | char(50)  | 否   |      | 标题     |     |
| link       | char(200) | 是   |      | 链接     |     |
| content    | text      | 否   |      | 内容     |     |
| category   | char(20)  | 否   |      | 类别     |     |
| createTime | dateTime  | 否   |      | 创建时间 |     |
